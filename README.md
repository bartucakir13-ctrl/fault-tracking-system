# Web-Based Fault Tracking System 🔧

A responsive, serverless web application designed for technical service centers to report, track, and archive hardware faults. This frontend project demonstrates client-side data management and dynamic UI rendering.

## 🚀 Features
* **Dashboard Interface:** Real-time statistics for pending, completed, and daily device registrations.
* **Session Management:** Simple login simulation using `sessionStorage` (Default User: Bartu / Pass: 12345).
* **Data Persistence:** Client-side database management utilizing the browser's `localStorage` API.
* **Live Search & Filter:** Instant filtering of pending devices by customer name, device model, or serial number.
* **Data Export:** One-click CSV export functionality for archived/completed repair records.
* **Responsive UI:** Custom CSS featuring modern gradients, a circuit-board texture background, animated toast notifications, and dynamic tables.

## 🛠️ Technologies Used
* **HTML5**
* **CSS3** (Custom properties, flexbox, grid, keyframe animations)
* **JavaScript** (ES6+, DOM Manipulation, Local Storage)
